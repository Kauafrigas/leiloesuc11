#leiloesuc11

#Este projeto visa criar um sistema informatizado para uma casa de leilões. O sistema permite o gerenciamento de itens de leilão, lances, e a organização de eventos de leilões de maneira eficiente e segura.

## Funcionalidades Principais
- Cadastro de itens para leilão
- Registro de lances
- Gestão de usuários e permissões
- Geração de relatórios de leilões

# Tecnologias utilizadas: MySQL e Java
